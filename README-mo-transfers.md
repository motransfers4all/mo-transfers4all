# ✦ MO Transfers4all Athens

**Live site:** [mo-transfers4all.gr](https://mo-transfers4all.gr)

Premium private transfer service in Athens by Marjus Oruci — airport transfers, city tours, port transfers and luxury transportation across Attica.

---

## Features

- **Luxury design** — dark navy & gold aesthetic, Cormorant Garamond typography
- **Services section** — airport, port, city tours, custom transfers
- **Fleet showcase** — vehicle cards with passenger capacity
- **Pricing table** — fixed routes with day/night rates
- **WhatsApp float button** — instant contact from any page
- **Cookie consent banner** — GDPR compliant
- **QR code page** — for print materials and business cards
- **SEO** — sitemap, robots.txt, JSON-LD TaxiService schema, OpenGraph
- **Legal pages** — Privacy Policy & Terms of Service

## Deployment

Hosted on **GitHub Pages** with a custom domain via CNAME.

To deploy:
1. Push changes to the `main` branch
2. GitHub Pages auto-publishes from the root directory
3. DNS points `mo-transfers4all.gr` to GitHub Pages via CNAME
